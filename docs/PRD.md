# 短视频漫剧生成器 - 产品需求文档

## 1. 产品概述

### 1.1 产品名称
**漫剧工坊 (ComicVideo Studio)**

### 1.2 产品定位
一款面向小白用户和编剧的 AI 辅助短视频漫剧创作工具，通过大模型 API 实现从网络小说到短视频的一站式生成。

### 1.3 目标用户
- 短视频创作者（无专业技能）
- 网络小说作者/编剧
- 自媒体运营者
- 漫画/动画爱好者

### 1.4 核心价值
- **降低门槛**：无需绘画、剪辑技能
- **提升效率**：AI 自动化生成流程
- **保持一致**：人物形象、场景风格统一

---

## 2. 功能架构

### 2.1 整体流程
```
网络小说 → 剧本生成 → 人物/场景设计 → 视频生成 → 导出发布
```

### 2.2 核心模块

| 模块 | 功能 | 调用模型 |
|------|------|----------|
| 剧本生成器 | 小说→分镜剧本 | GPT-4 / Claude |
| 人物设计器 | 生成角色立绘 | Midjourney / DALL-E 3 |
| 场景设计器 | 生成场景四视图 | Midjourney / Stable Diffusion |
| 视频合成器 | 图片→动态视频 | Runway / Pika Labs |

---

## 3. 详细功能设计

### 3.1 模块一：剧本生成器

**输入**：
- 网络小说文本（粘贴或上传 TXT）
- 选择章节范围
- 视频时长设定（30秒/1分钟/3分钟）

**处理流程**：
1. AI 分析小说内容，提取核心情节
2. 自动拆分为多个场景
3. 为每个场景生成：
   - 场景描述
   - 人物对话
   - 镜头指示（远景/近景/特写）
   - 情绪氛围

**输出**：
- 结构化剧本（JSON 格式）
- 可视化剧本预览
- 支持手动编辑调整

**剧本格式示例**：
```json
{
  "title": "第一集：初遇",
  "duration": "60s",
  "scenes": [
    {
      "id": 1,
      "description": "雨夜，古老的城门前",
      "characters": ["男主", "女主"],
      "dialogue": "你是谁？为何深夜独行？",
      "camera": "中景",
      "mood": "神秘、紧张"
    }
  ]
}
```

### 3.2 模块二：人物设计器

**输入**：
- 剧本中提取的人物列表
- AI 自动生成的人物描述
- 用户可补充/修改描述

**功能**：
1. **角色卡生成**：根据描述生成人物立绘
2. **多角度视图**：正面、侧面、背面、3/4 视角
3. **表情包生成**：喜、怒、哀、惊等表情
4. **服装变体**：不同场景的服装设计

**一致性保障**：
- 使用 LoRA 或 IP-Adapter 技术
- 保存人物种子参数
- 支持微调和重新生成

### 3.3 模块三：场景设计器

**输入**：
- 剧本中的场景描述
- 风格选择（古风/现代/科幻/奇幻）

**功能**：
1. **场景四视图生成**：
   - 全景图（建立空间感）
   - 中景图（人物活动区域）
   - 近景图（细节展示）
   - 氛围图（光影、天气）

2. **风格统一**：
   - 预设风格模板
   - 自定义风格参考图
   - 全局色调调整

### 3.4 模块四：视频合成器

**输入**：
- 人物立绘 + 表情
- 场景图片
- 剧本对话和镜头指示

**功能**：
1. **图生视频**：静态图片转动态
2. **口型同步**：对话配合口型动画
3. **镜头运动**：推拉摇移等效果
4. **转场效果**：场景切换动画
5. **配音配乐**：TTS 语音 + 背景音乐

**输出格式**：
- MP4 视频文件
- 支持 9:16（抖音）、16:9（B站）、1:1（小红书）

## 4. 用户界面设计

### 4.1 整体布局
- **顶部导航**：项目管理、设置、帮助
- **左侧步骤栏**：四个步骤进度指示
- **中间工作区**：当前步骤的操作界面
- **右侧预览区**：实时预览效果

### 4.2 交互原则
- 向导式流程，逐步引导
- 每步可返回修改
- 自动保存草稿
- 操作可撤销

---

## 5. 技术架构

### 5.1 前端技术栈
- **框架**：React 18 + TypeScript
- **UI 组件**：Ant Design / Shadcn UI
- **状态管理**：Zustand
- **视频预览**：Video.js

### 5.2 后端技术栈
- **框架**：Node.js + Express / Next.js API Routes
- **数据库**：PostgreSQL + Redis
- **文件存储**：阿里云 OSS / AWS S3

### 5.3 AI 模型集成
| 功能 | 推荐 API |
|------|----------|
| 文本理解/剧本 | OpenAI GPT-4 / Claude API |
| 图像生成 | Midjourney API / DALL-E 3 |
| 图生视频 | Runway Gen-3 / Pika Labs |
| 语音合成 | Azure TTS / 讯飞 |

---

## 6. 数据流设计

```
用户上传小说
    ↓
[剧本生成 API] → 结构化剧本 JSON
    ↓
[人物提取] → 人物描述列表
    ↓
[图像生成 API] → 人物立绘 + 场景图
    ↓
[视频生成 API] → 动态视频片段
    ↓
[合成导出] → 最终视频文件
```

## 10. 风险与对策

| 风险 | 对策 |
|------|------|
| 人物一致性差 | 使用 LoRA 训练 + 参考图 |
| 生成速度慢 | 异步队列 + 进度提示 |
| API 成本高 | 缓存复用 + 分级定价 |
| 版权问题 | 用户协议 + 内容审核 |
