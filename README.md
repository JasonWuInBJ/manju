# 🚀 快速开始
## 前提条件：
  安装 Docker Desktop 地址：<link>https://docs.docker.com/get-started/get-docker/</link>


## 执行安装：
  ```
  git clone https://github.com/saturndec/waoowaoo.git
  cd waoowaoo
  docker compose up -d
  ```

## 访问地址：
  <link>http://localhost:13000</link>
  或者 
  <link>http://localhost:30000</link>
